A Few Projects

Here are some of the projects I have worked on both in groups or by myself. Click on the image to be redirected to the live page or click "see code" to be taken to the code on github!

Sports Odds

A full-stack application that provides up-to-date sports news, game odds, current standings, and a platform for users to create an account and bet on games using fake money. It sources this information from multiple APIs and scrapes pages to ensure users have access to the latest information. Users can create an account to participate in the platform's betting features and enjoy a risk-free betting experience.

Fitness Tracker

A full-stack app that allows the user to track their recorded workouts in a journal, search for workouts that will displays a gif or image on how to perform workout and will generate a random motivational quote and track their recorded weights and display the change in a chart.

Admissions

A full-stack app that is used to assist with admissions. Made with seven other Software Engineers. In this application you can create candidates conduct a live coding interview with a shared coding editor and grade them on their performance, as well as select recommended study material for them. It also displays the information for all interviews your team has given and passing percentage.

AirBnb Clone

A full-stack application that was built to be a clone of the official Airbnb page. Made with four other Software Engineers.

Todo-List

My first time using Vue for a project. It allows the user to create a todo-list that sorts task by priority and time created. Saves list and user name utilizing local storage. Deployed on Vercel.

Memes

My first Front-End project. It makes API calls to imgflip and displays the top 100 meme templates and has links to direct you to either see some of the examples, or create your own with that template! There are a couple used in the page itself.

PC Health

This is a simple Python script that will give you information about your computers health such as CPU usage percentage, disk usage percentage, and available memory. It will also empty your recycling bin and delete some unwanted files or directories from your temp folder.
